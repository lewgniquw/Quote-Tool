## 二、项目定位

单文件 HTML 彩盒报价工具，415 行。面向浙江国立包装彩印部销售岗位，**纯手动输入**模式——刀版房给数字，销售填数字，计算器出报价。

**核心设计原则：**

1. **纯手动输入** — 面纸裁切尺寸、浪纸尺寸、拼版数全部手动填。
2. **费用全覆盖** — 面纸 + 浪纸 + 损耗 + 印刷 + 裱瓦 + 模切 + 粘盒 + 刀版 + 运输 + 表面处理
3. **定价公式** — 成本价 → ×(1+税率) → 含税价 → ÷(1-毛利率) → 最终报价
4. **详细计算过程** — 每道工序独立列出：单张价 → 元/个，便于逐道查错
5. **后台可调** — 密码 888888，表面处理/裱瓦/模切/粘盒/运输价格可改

## 四、计价公式详解

### 4.1 损耗计算

```
面纸：填了损耗张数 → (总张数+损耗)/总张数；留空 → 1.05（5%）
浪纸：填了损耗张数 → (总张数+损耗)/总张数；留空 → 1.03（3%）
```

### 4.2 面纸
```
平方单价 = 吨价 / (1,000,000 / 克重)
单张价  = 裁切面积(m²) × 平方单价 × 损耗系数
总价    = 单张价 × 总张数(数量÷拼版数)
裁切面积 = paperL × paperW / 1,000,000  (mm²→m²)
```

### 4.3 浪纸
```
计价面积 = 浪纸长 × (浪纸宽+20) / 1,000,000  (mm²→m²，宽+2cm是厂家要求)
单张价  = 计价面积 × 单价 × 损耗系数
```

### 4.4 印刷（阶梯定价）

**关键规则**：用总张数判断阶梯，不能用每千个张数（每千个张数永远 ≤1000）。

| 阶梯 | 120五色机 | 142五色机 |
|------|-----------|-----------|
| ≤1000 张 | 起印费 | 起印费 |
| ≤3000 张 | 起印费 | 起印费 |
| 3000~10000 张 | 起印费 + 阶梯价×(张数-3000) | 同 |
| >10000 张 | 超量单价×张数 | 同 |

各色数有独立起印价/阶梯价/超量价。下机费和改版费勾选才加。

### 4.5 裱瓦
```
单张价 = 面纸裁切面积(m²) × 0.18元/㎡
```

### 4.6 模切
```
装版费 + 加工费×总张数
```
不同机型装版费和加工费不同（35~50 + 0.08~0.30/张）。

### 4.7 粘盒
```
粘盒费 = 单价 × 数量（元/个）
```

### 4.8 刀版
```
刀版面积 = (浪纸长-10) × (浪纸宽-10) / 1,000,000  (mm²→m²)
新版费   = 刀版面积 × 200
旧版     = 0（免费）
```

### 4.9 运输
```
运输费 = 面纸面积 × 区域单价 × 总张数
```
按面纸裁切面积计算，7个区域独立定价。

### 4.10 表面处理
- **按面积（元/㎡）**：亮膜/哑膜/UV光油 + 亮油/哑油/UV油 → `单价 × 面纸面积 × 总张数`
- **按张（元/张）**：烫金、凹凸 → `单价 × 数量`

### 4.11 最终定价
```
allCost = 面纸 + 浪纸 + 印刷 + 裱瓦 + 模切 + 粘盒 + 刀版 + 运输 + 表面处理
cpu     = allCost / 数量                     （成本单价）
taxPrice = cpu × (1 + 税率)                  （含税单价）
sp      = taxPrice / (1 - 毛利率)             （最终报价）
total   = sp × 数量                          （总价）
```

**税率特殊处理**：`parseInt('0')` 在 JS 中是 falsy → 空字符串默认 13%，填 0 才是真不开票。

## 八、开发教训

1. **删了元素就要删代码引用** — 删掉 `paperSheets` 输入框后忘了删 `document.getElementById('paperSheets')`，JS 直接崩溃无反应。
2. **改函数返回值要同步改调用方** — `calcMount()` 从返回千个价改为返回单张价后，旧公式还在用千个价加总。
3. **印刷阶梯价用总张数不是每千个张数** — 每千个张数永远 ≤1000，导致任何数量都判到最低档。
4. **JS 中 `0` 是 falsy** — `parseInt('0')||13` 会把用户填的 0 税率覆盖为 13%。
5. **全局编号替换会误伤** — 概念页重编号时所有引用也会被改。
6. **改完先自检** — 修改报价器后必须用真实生产单数据跑一遍验算再告知用户。
