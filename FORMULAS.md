尺寸链

裁切长 = 刀版长 + 20mm
裁切宽 = Math.ceil((刀版宽+20)/50)×50  — 凑整5cm
浪纸长 = 刀版长 + 10mm
浪纸宽 = 刀版宽 + 30mm
总张数 = Math.ceil(数量/拼版数)

损耗

面纸损耗系数 = 填了张数 ? (总张数+损耗)/总张数 : 1.05
浪纸损耗系数 = 填了张数 ? (总张数+损耗)/总张数 : 1.03

分项计算

面纸: 吨价÷(1000000÷克重) × 裁切长×裁切宽÷1000000 × 损耗系数 = 单张价 × 总张数 = 分项总价
浪纸: 单价×(1-返点%÷100) × 浪纸长×浪纸宽÷1000000 × 损耗系数 = 单张价 × 总张数 = 分项总价
印刷: 张数≤3000→开机费 / >3000→开机费+超量单价×(张数-3000) / 免开机→超量单价×张数。结果×(数量÷1000)=分项总价
裱瓦: 0.20 × 裁切面积 × 总张数
模切: 装版费+加工费×总张数 / 免装版→跳过装版费
粘盒: 单价×数量
表面(按㎡): 单价 × 裁切面积 × 总张数
表面(按张): 单价×数量 + 压纹额外加¥100
刀版: 新版=刀版长×刀版宽÷1000000×250 / 旧版=0
运输: 区域单价 × 裁切面积 × 总张数
其他: 元/个=单价×数量 / 总价=直接金额 / 元/㎡=单价×裁切面积×总张数

汇总

总成本 = 面纸+浪纸+印刷+表面+裱瓦+刀版+模切+粘盒+运输+其他
成本单价 = 总成本÷数量
含税价 = 成本单价×(1+税率÷100)
含毛利 = 含税价÷(1-毛利率÷100)
总价 = 含毛利×数量


尺寸链

普通: 纸板长=(长+宽)×2+35 / 纸板宽=宽+高+4
全盖: 纸板长=(长+宽)×2+35 / 纸板宽=宽+高+宽
中开: 纸板长=高×2+长 / 纸板宽=宽+高+宽+高
天地盖: 纸板长=长+2×高 / 纸板宽=2×高+宽
异形: 手动输入

竖瓦: 纸板宽+20mm
横瓦: 长宽调换后新宽+20mm
损耗系数 = 填了损耗 ? (总张数+损耗)/总张数 : 1

分项计算

纸板: 纸板长×纸板宽÷1000000 × 单价×(1-返点%÷100) × 损耗系数 × 总张数
刀版: 纸板面积×250（需要时）
模切: 装版费+加工费×总张数（需要时）
印刷(传统): 上机费+单价×总张数
印刷(平方): 平方单价×纸板面积×总张数
柔版: 固定金额
糊口/打钉: 单价×数量
运输: 区域单价×纸板面积×总张数
其他: 同彩盒

汇总

总成本 = 纸板+刀版+模切+印刷+柔版+糊口+运输+其他
成本单价 = 总成本÷数量
含税价 = 成本单价×(1+税率÷100)
含毛利 = 含税价÷(1-毛利率÷100)
总价 = 含毛利×数量
